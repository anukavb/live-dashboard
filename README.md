# Live Collaboration Dashboard

A real-time collaborative workspace inspired by Notion/Figma.
Built for the Interactive Product Experience Platform challenge.

## Tech Stack
- React (Create React App)
- Zustand (global state)
- React Router DOM (routing)
- CSS Variables (theming)

## Features
- 3-panel layout: sidebar, workspace, activity feed
- Dark/light theme with localStorage persistence
- Command palette (Cmd+K)
- Live collaborator presence simulation
- Editable document blocks with progress tracking
- Comments system
- Keyboard shortcuts (T = theme, B = sidebar, Esc = close)

## Run Locally
npm install
npm start
